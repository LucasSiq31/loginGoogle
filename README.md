Este projeto é um simulador da interface de autenticação do Google, desenvolvido como parte da disciplina de **Design de Código** no curso de Engenharia de Software. O foco principal foi a implementação de lógica de validação de formulários.

---

### 🔑 Credenciais de Teste

Para interagir com o sistema e testar a lógica de autenticação, utilize os dados abaixo:

* **Usuário:** `user@email.com`
* **Senha:** `1234`

---

### 🚀 Funcionalidades

* **Validação de Acesso:** Lógica em JavaScript para verificar credenciais e permitir o fluxo de login.
* **Design Responsivo:** Adaptado para oferecer uma experiência consistente em dispositivos móveis e desktops.
* **Interatividade:** Feedback visual para campos obrigatórios e estados de erro.

### 🛠️ Tecnologias Utilizadas

A construção da interface e sua inteligência foram feitas com:

* **HTML5:** Estruturação dos formulários e elementos de entrada.
* **CSS3:** Estilização avançada para garantir a semelhança visual com o design original.
* **JavaScript:** Lógica de autenticação e manipulação de eventos de submissão.
